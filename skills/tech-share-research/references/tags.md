# 标签分类体系

## 技术类别

| 标签 | 说明 | 示例 |
|------|------|------|
| `AI 工具` | AI 编程/办公助手 | Cursor、Claude Code、QoderWork |
| `Agent` | 智能体/自动化 | Multi-Agent、AutoGen |
| `IDE` | 开发环境 | VSCode、JetBrains、Zed |
| `低代码` | 低代码/无代码平台 | - |
| `开发效率` | 提效工具/方法 | - |
| `产品调研` | 新产品/服务评估 | - |
| `技术趋势` | 行业趋势/方向 | - |

## 优先级

| 优先级 | 说明 | 响应时间 |
|--------|------|----------|
| `high` | 紧急，直接影响当前工作 | 24 小时内 |
| `medium` | 重要，本周内研究 | 1 周内 |
| `low` | 有兴趣，有空再看 | 按需 |

## 状态流转

```
backlog → researching → documented
                     ↘ abandoned
```

## 来源类型

- `技术分享会` - 线下/线上分享
- `文章` - 技术博客/新闻
- `讨论` - 团队讨论/社群
- `产品发布` - 官方发布
- `其他` - 其他来源
